# 模式
```
# Telegram, 0
31.13.86.0/22
74.86.235.0/24
91.108.56.0/23
91.108.56.0/22
91.108.4.0/22
109.239.140.0/24
149.154.164.0/22
149.154.167.0/24
149.154.168.0/22
149.154.172.0/22
```
- 其中的 `Telegram` 代表了规则名称
- `0` 则表示走代理
- `1` 则表示不走代理

那么上述规则，则表示为：
- 规则名：`Telegram`
- 下面的 IP CIDR 均走代理，不匹配的请求都绕过代理
